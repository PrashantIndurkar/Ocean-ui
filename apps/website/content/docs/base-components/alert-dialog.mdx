# Alert Dialog

<Description>
  A modal dialog for critical confirmations or destructive actions. Requires
  explicit dismissal (button or Escape); cannot be closed by clicking outside.
  Built with Ark UI Dialog (role="alertdialog") and Ocean UI design tokens.
</Description>
