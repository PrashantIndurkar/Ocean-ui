## Introduction

Welcome to **Ocean UI** - A collection of beautiful, copy-&-paste components for quickly building application UIs.

## Quick Start

Get started by installing Ocean UI in your project.

### Installation

```bash
npm install ocean-ui
```

### Usage

```tsx
import { Button } from "ocean-ui/components";

export function MyComponent() {
  return <Button>Click me</Button>;
}
```
