### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm
- TypeScript 5+

### Setup

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run development server
pnpm --filter @ocean-ui/website dev
```
